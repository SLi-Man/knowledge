# 系统服务管理

## 一、开机启动

文件位置：`/etc/rc.local`

作用：开机启动命令

第一次使用需要添加权限：`chmod +x /etc/rc.local`

---

## 二、运行级别 (Runlevels) 与目标 (Targets) 

### 2.1 传统 SysVinit 运行级别

- **0**: 关机 (halt)
- **1**: 单用户模式 (Single user mode, 救援模式)
- **2**: 多用户模式，不带网络 (不常用)
- **3**: 多用户模式，带网络 (**命令行界面，服务器常用**)
- **4**: 未使用
- **5**: 多用户模式，带网络和图形界面 (**桌面环境常用**)
- **6**: 重启

查看当前运行级别：

```bash
runlevel
```
