# 性能监控

## 一、/proc 文件系统

说明：`/proc` 是一个虚拟文件系统，提供了内核和进程的实时信息映射，是性能排查的第一手数据源。

### 1.1 文件对应

| 路径          | 说明                                                   |
| ------------- | ------------------------------------------------------ |
| /proc/cpuinto | 查看 cpu 的信息                                        |
| /proc/meminfo | 查看内存的信息                                         |
| /proc/loadavg | 查看负载的信息                                         |
| /proc/mounts  | 查看挂载的信息，唯一需要掌握的文件，其它都通过命令代替 |
