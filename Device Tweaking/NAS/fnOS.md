# 飞牛 OS

## 应用商城 HomeAssistant 手动更新

1. 打开docker的compose找到docker-home-assistantan点击停止
2. 在容器中删除assistantan（放心你的书记还在）
3. 在compose的yaml配置中把image改为 "ghcr.nju.edu.cn/home-assistant/home-assistant:stable"
4. 重新运行，将自动下载镜像并启动
